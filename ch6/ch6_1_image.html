Inserting image files for webpages.

การแทรกไฟล์รูปภาพสำหรับเว็บเพจ

Description
การแทรกรูปภาพ

สามารถทำได้โดย <img src="file location">

Description
การกำหนดขนาดของรูปภาพ

สามารถทำได้โดยใช้ Attribute เหล่านี้ width="ความกว้างรูปภาพ"
                                height = "ความสูงรูปภาพ"
                                alt = "ขอความกำกับภาพ"
ในส่วนของความสูงหรือความกว้างสามารถกำหนดเป็น px หรือ % ก็ได้
ความสูงของรูปนี้คือ 719px แต่เราสามารถใช้ hight=550px
เพื่อกำหนดขาดรูปภาพได้ โดยบราวเซอร์จะปรับ scale ให้เอง

Responsive Image

สามารถทำได้โดยใช้ style="width: 100%;height: auto;" หรือจะสร้าง class เองก็ได้

Warning
การใช้ width และ hight ร่วมกัน

บราวเซอร์ในปัจจุบันจะมีการปรับ scale รูปอัตโนมัติ เพราะฉนันเลือกแค่ว่าจะใช้ width หรือ height อย่างเดียวก็พอ ไม่งั้นอาจจะเป็นแบบรูปที่ 2 ได้
