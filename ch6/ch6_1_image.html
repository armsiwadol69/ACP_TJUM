Description
การกำหนดขนาดของรูปภาพและกำหนด

สามารถทำได้โดยใช้ Attribute เหล่านี้ width="ความกว้างรูปภาพ"
                                height = "ความสูงรูปภาพ"
                                alt = "ขอความกำกับภาพ"
ในส่วนของความสูงหรือความกว้างสามารถกำหนดเป็น px หรือ % ก็ได้
ความกว้างของรูปนี้คือ 1280px แต่เราสามารถใช้ width="50px"
เพื่อกำหนดขนาดรูปภาพได้ โดยบราวเซอร์จะปรับ scale ให้เอง

Warning
การใช้ width และ hight ร่วมกัน

บราวเซอร์ในปัจจุบันจะมีการปรับ scale รูปอัตโนมัติ เพราะฉนันเลือกแค่ว่าจะใช้ width หรือ height อย่างเดียวก็พอ ไม่งั้นอาจจะเป็นแบบรูปที่ 2 ได้ (อย่าหาทำ)
