Inserting audio files for webpages.

การแทรกไฟล์เสียงสำหรับเว็บเพจ

Example / Audio ; autoplay&controls (js volume control)

Description

การแทรกไฟล์เสียง

สามารถทำได้โดย
<audio src="ที่อยู่ของไฟล์"></audio>

Attributes

autoplay = เสียงจะเล่นอัตโนมัติเมือเพจโหลดเสร็จ
controls = แสดงแผงควบคุมการเล่น
loop = เมื่อเล่นจบจะเล่นซ้ำอัตโนมัติ
muted = ปิดเสียงเมื่อเล่น (แต่สามารถเปิดได้)

File Formatt Support

MP3 audio/mpeg
OGG audio/ogg
WAV audio/wav
