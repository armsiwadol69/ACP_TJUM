Inserting video files for webpages.

การแทรกไฟล์วีดีโอสำหรับเว็บเพจ

Example / Video | Javascript Delay 4.20sec. Before Start Playing

Description

การแทรกวีดีโอ

สามารถทำได้โดย
<video src="ที่อยู่ของไฟล์"></video>

Attributes

autoplay = วีดีโอจะเล่นอัตโนมัติเมือเพจโหลดเสร็จ
controls = แสดงแผงควบคุมการเล่น
loop = เมื่อเล่นจบจะเล่นซ้ำอัตโนมัติ
poster = แสดงรูปก่อนที่จะเริ่มเล่น/ยังโหลดไม่เสร็จ
muted = ปิดเสียงเมื่อเล่น (แต่สามารถเปิดได้)
width / height = กำหนดความกว้าง หรือ ความสูงของวีดีโอ

Responsive Video
