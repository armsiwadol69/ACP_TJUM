Description

การแทรกวีดีโอจากแหล่งภายนอก (Embed/iframe)

สามารถทำได้โดย
<iframe src="ลิงค์วีดีโอ"></iframe>
กำหนดขนาดของ player โดยใช้ Attributes hight / width

Parameters (ex. URL?autoplay=1&loop=1)

&autoplay = เสียงจะเล่นอัตโนมัติเมือเพจโหลดเสร็จ
&controls=0,1 = แสดงแผงควบคุมการเล่น
&loop=0,1 = เมื่อเล่นจบจะเล่นซ้ำอัตโนมัติ
&mute=0,1 = ปิดเสียงเมื่อเล่น (แต่สามารถเปิดได้)

Responsive Video Player

อยากให้วีดีโอปรับขนาดตามขนาดหน้าจอก็สามารถทำได้โดย
style="width: 100%;height: auto;"
W/H = Auto อย่างใดอย่างหนึ่งหรือจะสร้าง class เองก็ได้
