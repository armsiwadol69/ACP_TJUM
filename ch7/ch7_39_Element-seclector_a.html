Class and ID Selector

การใช้ Class Selector ---> การใช้ Selectors แบบ Class การประกาศเริ่มต้นด้วย (.) ตามด้วยชื่อ Class โดยชื่อ Class สามารถเป็นตัวอักษร ตัวเลข เครื่องหมาย -,_ เท่านั้น ห้ามมีช่องว่าง วิธีการใช้งาน - Class Selectors เหมาะกับการใช้งานเมื่อต้องการแสดงผลรูปแบบนั้นซ้ำหลาย ๆ ครั้งกับ elements หลาย ๆ อัน
การใช้ ID Selector ---> การใช้ Selectors แบบ ID การประกาศเริ่มต้นด้วย (#) ตามด้วยชื่อ ID โดยชื่อ ID สามารถเป็นตัวอักษร ตัวเลข เครื่องหมาย -,_ เท่านั้น ห้ามมีช่องว่าง วิธีการใช้งาน - ID Selectors เหมาะกับการใช้งานเมื่อต้องการแสดงผลกับ element เพียงตัวเดียว เนื่องจาก ID เป็น unique สามารถมีได้ 1 จุดต่อ document ถ้ามีการประกาศซ้ำจะแสดงผลเพียงครั้งเดียว
