Paragraph Break เป็นแท็กแสดงการขึ้นย่อหน้าใหม่

HTML มีโครงสร้างการเขียนโดยอาศัย Tag ในการควบคุมการแสดงผลของข้อความรูปภาพ หรือวัตถุอื่นๆ แต่ละ Tag อาจจะมีส่วนขยาย เรียกว่า Attribute สำหรับจัดรูปแบบเพ่มเติม

     Attribute เป็นส่วนขยายใน tag ใช้สำหรับจัดรูปแบบเพิ่มเติม เช่น ขนาด สี ระยะห่าง เป็นต้น ค่าของ attribute จะอยู่ในเครื่องหมาย "...."

     Comment หรือข้อความหมายเหตุ มีประโยชน์หลายอย่าง เช่น ใช้เพื่อเตือนความจำ กำกับ code แต่ละส่วนที่เราเขียนว่าเพื่อทำอะไร ทำให้อ่าานและแก้ไข code ภายหลังได้ง่าย ข้อความใน tag comment จะไม่ถุกเว็บบราวเซอร์แสดงออกมาให้ผู้ใช้งานได้เห็น จะเห็นเมื่อมีการ view source code เท่านั้น
